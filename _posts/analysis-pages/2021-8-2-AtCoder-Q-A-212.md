### 题目大意

> 给 A B, 判断是哪种金属
>
> 如果 A>0 并且 B=0    -> Gold
>
> 如果 A=0 并且 B>0    -> Silver
>
> 如果 A>0 并且 B>0    -> Alloy

### 思路

> 直接做条件判断
>
> 复杂度：O(1)

### 代码

~~~java
import java.util.*;
public class Main {
   public static void main (String [] args){
       Scanner in=new Scanner(System.in);
       int A = in.nextInt();
       int B = in.nextInt();
       if(A > 0 && B == 0) System.out.println("Gold");
       if(A == 0 && B > 0) System.out.println("Silver");
       if(A > 0 && B > 0) System.out.println("Alloy");
   } 
}
~~~
