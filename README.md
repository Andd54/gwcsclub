## 我们是谁

我们是广外 CS 社，我们希望为所有对计算机科学感兴趣的人建立一个共同的社区。在这个社区中，我们分享自己的知识，互相帮助，互相学习。

我们欢迎所有想从0开始学习编程或者继续精进自己编程技术的人。我们并不局限于特殊的几种编程语言与竞赛，如果你想做任何有趣的编程项目，我们欢迎你将自己的代码仓库链接展示在我们的网页上。
